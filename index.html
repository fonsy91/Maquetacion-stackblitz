          <section class="container">
            <div class="box">Caja 1</div>
            <div class="box">Caja 2</div>
            <div class="box">Caja 3</div>
          </section>
        

          /* Cada caja ocupa toda la línea */
          .box {
            display: block;
          }
      
          /* Las cajas se alinean horizontalmente pero sin 
          margen vertical */
          .box {
            display: inline;
          }
      
          /* Como inline pero respeta ancho, alto y margen */
          .box {
            display: inline-block;
            width: 100px;
          }
      
          .container {
            display: flex; /* Las cajas se alinean en fila por defecto */
            gap: 10px; /* Espacio entre cajas */
          }
      
          .container {
            display: grid;
            /* 3 columnas iguales */
            grid-template-columns: repeat(3, 1fr);
            gap: 10px;
          }

Hay que diferenciar bien las propiedades dde justify-content (horizontal) y align-content (vertical)

  1. Justify-content: start, center, end, space-between, spaca-around, space-evenly
  2. Align-items: start, center, end, stretch, baseline
  3. align-content: start | end | center | space-between | space-around | space-evenly | stretch

Tambien podemos usar repeat() en vez de grid-template-columns y grid-template-rows ya que puede ser repetitivo y molesto.

grid-template-columns: 100px repeat(4, 50px) 200px; que equivale a grid-template-columns: 100px 50px 50px 50px 50px 200px; 


          <section class="contenedor-grid">
            <div>Primera celda</div>
            <div>Segunda celda</div>
            <div>Tercera celda</div>
            <div>Cuarta celda</div>
            <div>Quinta celda</div>
          </section>
        

          .contenedor-grid {
            display: grid;
            grid-template-columns: 150px 150px 150px;
            grid-template-rows: 150px 150px 150px;
            column-gap: 10px;
            row-gap: 15px;
          }

          /* Ponemos cada hijo de un color para distinguirlos */
          .contenedor-grid :nth-child(1) { background: indigo; }

          .contenedor-grid :nth-child(3) { background: green; }

-auto-fill reserva espacio aunque no haya contenido.
-auto-fit hace que las columnas vacías desaparezcan y el contenido se estire.

- Si el contenedor es mas ancho, se agregan mas columnas de 150px
- Si sobra espacio, quedan "columnas vacias" al final de la fila.
- Si se hace mas pequeño se va adaptando a la pantalla y la columna da un salto de linea, se puede traducir "auto-fill" = "crea tantas columnas como quepan en el contenedor". Entonces si traducimos esto: repeat(auto-fill, minmax(150px, 1fr)) seria:

  1. Crea tantas columnas de al menos 150px como quepan en el contenedor.
  2. Cada columna puede estirarse hasta llenar el espacio restante (1fr) fr = (fracción restante).
  3. Si sobra espacio, con auto-fill podrían aparecer columnas vacías.
