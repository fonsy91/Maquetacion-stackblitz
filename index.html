PRÁCTICA DEL DISPLAY


          <section class="container">
            <div class="box">Caja 1</div>
            <div class="box">Caja 2</div>
            <div class="box">Caja 3</div>
          </section>
        

          /* Cada caja ocupa toda la línea */
          .box {
            display: block;
          }
      
          /* Las cajas se alinean horizontalmente pero sin 
          margen vertical */
          .box {
            display: inline;
          }
      
          /* Como inline pero respeta ancho, alto y margen */
          .box {
            display: inline-block;
            width: 100px;
          }
      
          .container {
            display: flex; /* Las cajas se alinean en fila por defecto */
            gap: 10px; /* Espacio entre cajas */
          }
      
          .container {
            display: grid;
            /* 3 columnas iguales */
            grid-template-columns: repeat(3, 1fr);
            gap: 10px;
          }

Por defecto todos los contenedores HTML tiene la propiedad display: block
